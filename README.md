Site d'achat et vente en ligne basé sur ricardo.ch

## Démarrage serveur
1. Ouvrir la console dans le dossier de `manage.py`
2. Lancer `python manage.py runserver`
3. Ouvrir localhost:8000

## Migrations
Commande : `python manage.py migrate`
